## Embedding Unity WebGL Builds

For more complex game logic or physics simulations that rely heavily on the Unity Engine, you can export your Unity project to WebGL and embed it using the `UnityEmbed` Astro component.

Below is an example of what that embed looks like. *(Note: The `src` here is a placeholder. To embed a real Unity game, you would drop the built WebGL folder into the `public/` directory and set the `src` to the `index.html` file of the build).*

```astro
<UnityEmbed src="/path/to/your/unity-build/index.html" height="500px" title="My Unity Game" />
```

## Why a Hybrid Approach?

- **R3F / Three.js**: Ideal for subtle page interactions, UI flourishes, and lightweight physics (using `@react-three/cannon`). It integrates perfectly with your React/Tailwind state.
- **Unity WebGL**: Perfect for porting existing museum installations, complex VR prototypes, or heavy 3D games where rebuilding in Three.js would be too time-consuming.
