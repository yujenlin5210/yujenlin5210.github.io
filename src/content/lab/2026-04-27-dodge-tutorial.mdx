<TutorialChapter title="1. The Canvas Setup">
  <div slot="text">
    Every great game starts with a blank canvas. 
    
    In React, we need a way to reference the underlying DOM element to draw on it. We use the `useRef` hook to grab the `<canvas>` element once it mounts.

    ```javascript
    import { useRef, useEffect } from 'react';

    export default function Game() {
      const canvasRef = useRef(null);

      useEffect(() => {
        const canvas = canvasRef.current;
        const ctx = canvas.getContext('2d');
        
        // 1. Draw the background
        ctx.fillStyle = '#0f172a';
        ctx.fillRect(0, 0, canvas.width, canvas.height);
      }, []);

      return <canvas ref={canvasRef} width={800} height={800} />;
    }
    ```

    Next, we draw the "Ship". To draw a triangle in Canvas, we move the pen to a starting point using `moveTo`, draw lines to the base corners with `lineTo`, and fill or stroke the path.

    ```javascript
    // 2. Draw the Ship
    ctx.save();
    ctx.translate(canvas.width / 2, canvas.height / 2); // Move to center
    ctx.rotate(-Math.PI / 2); // Point upwards
    
    ctx.beginPath();
    ctx.moveTo(20, 0); // Tip
    ctx.lineTo(-10, 10); // Bottom right
    ctx.lineTo(-10, -10); // Bottom left
    ctx.closePath();
    
    ctx.strokeStyle = '#818cf8';
    ctx.lineWidth = 4;
    ctx.stroke();
    
    ctx.restore();
    ```

<TutorialChapter title="2. Player Controls">
  <div slot="text">
    To make our ship move, we need to track keyboard input and update the ship's position. In React, we use a `useRef` to store the player's position and velocity. We don't use `useState` because we update these values 60 times a second inside `requestAnimationFrame`, and `useState` would trigger too many expensive React re-renders.

    We attach event listeners to track the `keydown` and `keyup` events. To prevent the browser page from scrolling when you press the arrow keys or space, we call `e.preventDefault()`:

    ```javascript
    const playerRef = useRef({ 
      x: 200, y: 200, 
      vx: 0, vy: 0, 
      angle: -Math.PI / 2 
    });
    const keysRef = useRef({});

    useEffect(() => {
      const down = (e) => {
        if (['Space', 'ArrowUp', 'ArrowDown', 'ArrowLeft', 'ArrowRight'].includes(e.code)) {
          e.preventDefault();
        }
        keysRef.current[e.code] = true;
      };
      const up = (e) => keysRef.current[e.code] = false;
      
      window.addEventListener('keydown', down);
      window.addEventListener('keyup', up);
      // cleanup...
    }, []);
    ```

    Inside the game loop, we apply acceleration to our velocity (`vx` and `vy`) based on the active keys. Instead of directly changing `x` and `y`, using velocity allows us to apply a bit of friction (`vx *= 0.95`) so the ship slides smoothly to a stop, giving it a cool, slippery asteroid-style feel. 
    
    Finally, we rotate the ship to face its direction of movement using `Math.atan2(vy, vx)`.

<TutorialChapter title="3. Enemies & Movement">
  <div slot="text">
    A dodge game needs something to dodge. Let's create a swarm of enemies.
    
    We store our dots in an array inside a `useRef`. In a real game, you might spawn them outside the screen and move them inwards, but for now we'll just scatter them randomly across the canvas with a random starting velocity.

    ```javascript
    const dotsRef = useRef([]);

    // Initialize 20 dots
    for (let i = 0; i < 20; i++) {
      dotsRef.current.push({
        x: Math.random() * 400,
        y: Math.random() * 400,
        vx: (Math.random() - 0.5) * 2,
        vy: (Math.random() - 0.5) * 2,
        radius: 3,
        color: '#ffffff'
      });
    }
    ```

    Inside the `requestAnimationFrame` loop, we iterate through the `dotsRef.current` array. For each dot, we update its position by adding its velocity (`x += vx`). Then, we render it as a small circle using the Canvas `arc()` path method. 
    
    If a dot moves off the edge of the screen, we don't delete it. Instead, we simply wrap its coordinates to the opposite side of the canvas. This is an old arcade trick that keeps the screen permanently populated without having to run an expensive spawning/despawning loop.

<TutorialChapter title="4. Gravity & Physics">
  <div slot="text">
    The unique mechanic of Dodge is that the player acts like a magnet. The closer an enemy gets to the ship, the stronger the pull. We achieve this by calculating the distance between the player and each dot using the Pythagorean theorem (`a² + b² = c²`).

    If the distance is within our `GRAVITY_RADIUS`, we calculate an inverse-square-style force: the smaller the distance, the greater the force. 

    ```javascript
    const GRAVITY_RADIUS = 150;
    const GRAVITY_STRENGTH = 0.045;

    dots.forEach(dot => {
      const dx = player.x - dot.x;
      const dy = player.y - dot.y;
      const dist = Math.sqrt(dx * dx + dy * dy);

      if (dist < GRAVITY_RADIUS) {
        // Calculate an inverse force (0 to 1 scale)
        const force = ((GRAVITY_RADIUS - dist) / GRAVITY_RADIUS) * GRAVITY_STRENGTH;
        
        // Apply force along the normalized directional vector
        dot.vx += (dx / dist) * force;
        dot.vy += (dy / dist) * force;
        
        // Draw a tether line for visual feedback
        ctx.beginPath();
        ctx.moveTo(player.x, player.y);
        ctx.lineTo(dot.x, dot.y);
        ctx.strokeStyle = "rgba(99,102,241," + (0.6 * (GRAVITY_RADIUS-dist)/GRAVITY_RADIUS) + ")";
        ctx.stroke();
      }
    });
    ```

    Notice how we divide `dx` and `dy` by `dist`. This normalizes the vector so it points exactly at the player with a length of 1, allowing us to safely multiply it by our `force`.
    
    This creates a slingshot effect. If you stand still, enemies will cluster and crash into you. If you keep moving, you can whip them behind you in an orbit!

<TutorialChapter title="5. Collision & Polish">
  <div slot="text">
    To make it a real game, we need collision detection. 

    Because our ship and the enemy dots are essentially circles from a hit-box perspective, we can use simple circle-circle collision. We already calculate the distance between the ship and each dot for gravity. All we have to do is check if that distance is less than the sum of their radii!

    ```javascript
    // Inside the dots loop
    const hitRadius = 12; // approximate radius of our ship

    if (dist < hitRadius + dot.radius) {
      // Collision detected!
      setGameState('GAMEOVER');
      crashed = true;
    }
    ```

    When a crash occurs, we set a `gameState` variable (like `'PLAYING'` or `'GAMEOVER'`). If we crash, we stop calling `requestAnimationFrame`, halting the game loop entirely. We conditionally render a crash screen overlay in React, and add logic to restart the game state when the user hits SPACE.
    
    *Note on Game Architecture:* The game stays inside one React component, but the important behavior is split into small helper functions like `updatePlayer`, `drawPlayer`, and `updateAndDrawDots`. That keeps the main loop readable without implying we moved everything out into separate modules.

    With that, the core mechanics of Dodge are complete! You can enhance the game further by adding a score timer, high score tracking, particle explosion effects when you crash, or mobile touch controls.

    statusRef.current = 'PLAYING';
    setGameState('PLAYING');
    
    if (rafRef.current) cancelAnimationFrame(rafRef.current);
    
    const ctx = canvas.getContext('2d');
    
    const loop = () => {
      if (statusRef.current !== 'PLAYING') return;

      ctx.fillStyle = '#0f172a';
      ctx.fillRect(0, 0, canvas.width, canvas.height);

      updatePlayer(pRef.current, kRef.current, canvas.width, canvas.height);
      const crashed = updateAndDrawDots(ctx, dotsRef.current, pRef.current, canvas.width, canvas.height);
      drawPlayer(ctx, pRef.current);

      // [NEW] If crashed, halt the loop and display the Game Over UI
      if (crashed) {
        statusRef.current = 'GAMEOVER';
        setGameState('GAMEOVER');
      } else {
        rafRef.current = requestAnimationFrame(loop);
      }
    };
    rafRef.current = requestAnimationFrame(loop);
  };

  return (
    <>
      <canvas ref={canvasRef} width={400} height={400} tabIndex={0} />
      {gameState !== 'PLAYING' && (
        <div style={{
          position: 'absolute', top: 0, left: 0, right: 0, bottom: 0,
          background: 'rgba(0,0,0,0.8)', color: 'white',
          display: 'flex', flexDirection: 'column', 
          alignItems: 'center', justifyContent: 'center'
        }}>
          <h2>{gameState === 'START' ? 'DODGE' : 'CRASHED'}</h2>
          <p>Press SPACE to {gameState === 'START' ? 'Start' : 'Retry'}</p>
        </div>
      )}
    </>
  );
}
